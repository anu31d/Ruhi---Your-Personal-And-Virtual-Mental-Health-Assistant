# **App Name**: RUHI: Your AI Companion

## Core Features:

- AI-Powered Chatbot: Engage in real-time conversations with an AI chatbot that provides empathetic and non-judgmental support using the OpenAI API.
- Guided Breathing Exercises: Access animated breathing exercises with visual timers and calming UI to help regulate stress.
- Mindfulness Prompts: Explore short, guided mindfulness prompts and daily affirmations to promote emotional grounding.
- User Authentication: Securely authenticate users with Firebase Authentication, supporting email and Google OAuth.
- Session History: Maintain a non-detailed, privacy-safe history of user sessions stored in Firestore.
- Therapist Style Tool: AI tool with access to background knowledge, retrieves relevant contextual information and incorporates that in its responses to the user.
- Crisis Safe Responses: Detect trigger words and offer resources, such as the suicide prevention hotline number. Does not provide a recommendation, but will include a call to action, or encourage the user to take specific actions.

## Style Guidelines:

- Primary color: Soft Lavender (#D0C6E1) to promote relaxation and tranquility.
- Background color: Very light gray (#F5F5F5) for a clean and unobtrusive backdrop.
- Accent color: Muted Blue (#A0B4CF) for interactive elements and highlights, providing a calming contrast.
- Body and headline font: 'PT Sans' (sans-serif) provides a modern yet approachable feel for both headlines and body text.
- Use gentle, rounded icons related to mental health and well-being.
- Incorporate slow, intentional animations and micro-interactions for a calming and engaging user experience.
- Employ a minimal text density with rounded components to foster trust and softness in the design.
